Projeto Pixels Art


Este projeto foi desenvolvido em HTML, CSS e javascript com requisitos obrigatrios. Foi desenvolvido uma página para criar uma matrix de 'pixels' em branco que podem ser coloridos pelo usuário.

A matriz pode ter o tamanho definido pelo usuário, que seleciona uma cor na paleta de cores e, ao clicar nos quaradinhos em branco ele é preenchido com a cor escolhida.

Projeto To do List


Este projeto foi desenvolvido em HTML, CSS e javascript ao mesmo tempo do pixels art. A página consiste em ser um criador de lista de tarefas.

O usuário escreve uma tarefa que precisa realizar, adiciona à lista e depois de cumprida pode apagar ou marcar como completa. Os dados desta lista ficam salvos no navegador, portanto o usuário pode fechar e abrir o navegador novamente que os dados não serão perdidos.

Projeto Facebook Signup


Este projeto foi desenvolvido em HTML, CSS e javascript em dupla. O objetivo foi criar uma pagina igual a de cadastro do facebook..

O usuário preenche os dados necessários no formulário e depois clica em castre-se. Após clicar em cadastrar, os dados coletados no formulário são exibidos na página, como forma de simular o envio das informações para um procesamento de dados.
